Semana 1
Guía de ejercicios y aplicación práctica
Objetivo de la semana
Comprender la relación entre ciudadanía digital, uso responsable de la información, pensamiento algorítmico y organización técnica del trabajo, iniciando el uso de Git y GitHub para registrar y publicar la actividad realizada.

Entrega final
Al finalizar la semana, el estudiante debe tener un repositorio publicado en GitHub con las respuestas y archivos de la actividad, y debe compartir el enlace del repositorio en Moodle.

Instrucciones generales
Responde las preguntas de la actividad.
Guarda todas las respuestas en archivos dentro de la carpeta de la semana.
Publica la carpeta y los archivos en un repositorio GitHub.
Comparte en Moodle el enlace del repositorio.
Pregunta 1 — Selección múltiple
¿Cuál de las siguientes opciones describe mejor qué es la ciudadanía digital?

A. El uso de internet únicamente para entretenimiento
B. El uso responsable de la tecnología para informarse, participar y compartir información de manera ética
C. El uso de redes sociales sin restricciones
D. El uso exclusivo de software comercial
Pregunta 2 — Selección múltiple
¿Cuál es la razón principal por la que es importante verificar la información antes de compartirla en internet?

A. Para que la publicación tenga más reacciones
B. Para evitar consumir más datos móviles
C. Para evitar difundir información falsa o engañosa
D. Para que la publicación sea más extensa
Pregunta 3 — Selección múltiple
¿Cuál de las siguientes opciones define mejor un algoritmo?

A. Una aplicación móvil
B. Una lista de pasos ordenados para resolver un problema
C. Una carpeta del computador
D. Una red social
Pregunta 4 — Reflexión de la semana
En un texto de 8 a 10 líneas, responde:

¿Qué relación encuentras entre el uso responsable de la información en internet, la participación ciudadana y la organización del trabajo técnico con herramientas como Git?
Pregunta 5 — Actividad práctica de algoritmia
Escribe un algoritmo de mínimo 6 pasos para explicar:

Cómo compartir información confiable en internet.

Ejemplo de inicio:

buscar la información
revisar la fuente
confirmar la noticia
Completa el algoritmo hasta tener al menos 6 pasos.

Estructura de archivos de la actividad
Dentro de tu proyecto crea esta carpeta:

semana1
Dentro de semana1 crea los siguientes archivos:

preguntas-semana1.txt
reflexion-semana1.txt
algoritmo-semana1.txt
Contenido de los archivos
1. Archivo: preguntas-semana1.txt
En este archivo debes escribir:

la Pregunta 1 con tu respuesta
la Pregunta 2 con tu respuesta
la Pregunta 3 con tu respuesta
Ejemplo de formato:

Pregunta 1:
Respuesta: B

Pregunta 2:
Respuesta: C

Pregunta 3:
Respuesta: B
2. Archivo: reflexion-semana1.txt
En este archivo debes escribir la respuesta de la Pregunta 4.

3. Archivo: algoritmo-semana1.txt
En este archivo debes escribir la respuesta de la Pregunta 5.

Crear cuenta y repositorio en GitHub
Si aún no tienes cuenta, crea una en:

https://github.com

Luego crea un repositorio público llamado:

programadores-para-la-paz
Publicar la actividad en el repositorio
Después de crear los archivos y responder las preguntas, debes publicar la actividad en tu repositorio.

Comandos sugeridos:


git add .
git commit -m "Actividad semana 1"
git push 
Verificación final
Antes de entregar, revisa que tu repositorio tenga publicada la carpeta semana1 con estos archivos:

preguntas-semana1.txt
reflexion-semana1.txt
algoritmo-semana1.txt
Entrega en Moodle
En Moodle debes publicar:

el enlace de tu repositorio GitHub
confirmación de que allí están publicadas las respuestas y archivos de la Semana 1
